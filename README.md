<h1 align="center">欢迎使用 h5ve-design</h1>

## 安装
1. 下载代码
```sh
git clone https://github.com/h5ve/h5ve-design.git

git clone https://gitee.com/h5ve/h5ve-design.git
```
2. 进入项目目录
```sh
cd h5ve-design
```

3. 安装依赖包
```
npm install
```
#### 无法安装请使用腾讯镜像
```
npm install --registry=http://mirrors.cloud.tencent.com/npm/ 
```

## 使用

本地启动应用
```sh
npm run serve
```

## 打包

```sh
npm run build
```
